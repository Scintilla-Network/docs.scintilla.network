# About

Scintilla-Network is a protocol for a decentralized, modular, and scalable network.

It is designed to be a general-purpose network that can be used for a wide variety of applications, including but not limited to: social media, file sharing, and web hosting.

The protocol is designed to be modular, so that it can be easily extended to support new features and applications. It is also designed to be scalable, so that it can support a large number of users and applications.

It's Identity system allows it to be specifically suited for RWA (Real World Assets) and DeFi (Decentralized Finance) applications.
